Address Autocomplete

The purpose of this webpage is to demonstrate a typeahead use case for the Lightbox Geocoder Autocomplete API.

Please enter your Lightbox Developer Portal API key. This will be used to pull data from the autocomplete API key.

While entering a partial address, the LBX geocoder will return a set of address candidates based on autocompletion of the partial string 'text', and display those results as a select dropdown.
